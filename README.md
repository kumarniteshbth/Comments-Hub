# Comments-Hub
**Comment Hub** is a full-stack CRUD web application built with **Node.js, Express.js, MongoDB, Mongoose, and EJS**. It enables users to create, view, edit, delete, and like comments with a responsive interface. The project implements RESTful routing, session-based Facebook-style Like/Unlike functionality, and efficient MongoDB data management.


## Comment Hub – Project Description

**Comment Hub** is a full-stack CRUD web application developed using **Node.js, Express.js, MongoDB, Mongoose, and EJS**. It allows users to create, view, edit, delete, and like comments through a clean, responsive, and modern interface inspired by social media platforms.

The application follows the **MVC architecture** and stores all comment data in **MongoDB** using **Mongoose** for efficient database operations. It also includes a **Facebook-style Like/Unlike** feature powered by Express sessions, ensuring each user can like or unlike a comment only once per session.

### Key Features

* Create new comments
* View all comments
* View individual comment details
* Edit existing comments
* Delete comments
* Facebook-style Like/Unlike functionality
* MongoDB database integration using Mongoose
* Session-based like tracking
* Responsive and modern UI
* Custom 404 error page
* RESTful routing with Method Override
* Dynamic pages rendered using EJS templates

### Technologies Used

* **Frontend:** HTML5, CSS3, EJS
* **Backend:** Node.js, Express.js
* **Database:** MongoDB, Mongoose
* **Other:** Express Session, Method Override

This project demonstrates the implementation of **CRUD operations**, **database connectivity**, **RESTful APIs**, **server-side rendering**, and **session management** while following modern web development practices. It serves as a practical example of building a complete full-stack web application with a user-friendly interface and efficient data management.
